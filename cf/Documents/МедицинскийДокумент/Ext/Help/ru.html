Медицинский документ

Документ Медицинский документ предназначен для сохранения медицинского документа.

В реквизиты документа и в реквизиты его табличных частей записывается оперативная информация, необходимая для реализации бизнес-процессов, регистрации выполнения медицинских услуг.

В табличных частях документа Медицинский документ записываются данные, которые формируются с помощью обработки "Формирование медицинских документов". В табличные части вынесены данные, которые относятся к управлению бизнес-процессами и используются для формирования регламентированной отчетности.

  • Табличная часть Выполненные услуги служит для регистрации факта выполнения простых медицинских услуг, в том числе и простых услуг, которые выполнялись в составе сложных услуг.
  • Табличная часть Назначенные услуги служит для регистрации услуг, назначенных врачом во время выполнения простой медицинской услуги.
  • Табличная часть Исполнители служит для регистрации исполнителей медицинских услуг (врачей, средних и младших медицинских работников), зарегистрированных в табличной части Выполненные услуги.
  • Табличная часть Места выполнения служит для регистрации медицинских кабинетов, палат и прочих мест, где производилось выполнение медицинских услуг, зарегистрированных в табличной части Выполненные услуги.
  • Табличная часть Диагнозы служит для регистрации выявленных диагнозов при выполнении медицинской услуги.
  • Табличная часть Оборудование служит для регистрации медицинского оборудования, с использованием которого выполнялась услуга.
  • Табличная часть Лекарственные назначения служит для регистрации лекарственных препаратов, которые были назначены пациенту.
  • Табличная часть Показатели здоровья служит для отображения информации о показателях здоровья, которые были зарегистрированы в медицинском документе.

Реквизиту Тело медицинского документа присваивается XML файл, который был сформирован шаблоном медицинского документа (ШМД).
